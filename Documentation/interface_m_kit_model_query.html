- (NSDictionary *) execute: (NSError **)  error

Executes the query and returns a dictionary. The dictionary has two keys: totalCount and results. totalCount contains the total number of items the query matches, results is an array of objects that match the query. Note that the number of items returned might be limited by the backend service and therefore, totalCount and number of objects in the results array can be different.

Parameters
errorThe error if any
Returns
The results of the query.
